# New2Lab

Welcome to the Amaro Lab. This page contains basic information that is valuable to know when joining the lab. Please read the entire page and complete the tutorials at the bottom. These will teach you basics skills that are essential for your work here.

## Progress Reports[edit](</mediawiki/index.php?title=New2Lab&action=edit&section=5> "Edit section: Progress Reports")]

Progress reports are turned in every 1st of the month in a Dropbox folder that you will be invited to join (ask a senior lab member to invite you). 

The reports should contain a small summary of the work you have been doing over the month, the format is not strict. Look at other lab members reports for examples.

### Tutorials[edit](</mediawiki/index.php?title=New2Lab&action=edit&section=12> "Edit section: Tutorials")]

Some of these tutorials overlap with the tutorials in the Rotation Student Conspectus. 

Before starting each tutorial, remember to download all of the required tutorial input files (and, in some cases, the output files/results) 

  * [[1]](<https://ctlee.github.io/BioChemCoRe-2018>). This Github tutorial for MD was used for BioChemCoRe in 2018.
  * [Autodock4 with AutodockTools](<http://autodock.scripps.edu/faqs-help/tutorial/using-autodock-4-with-autodocktools>). If you will be performing Molecular Docking, you might find these tutorials helpful
  * [Amber Tutorials](<http://ambermd.org/tutorials/>). Software used for setting up and running molecular dynamics simulations (Please complete the B1-B4 tutorials)
  * [NAMD Tutorials](<http://www.ks.uiuc.edu/Training/Tutorials/namd-index.html>). Software used for running molecular dynamics simulations. Please complete the [Main NAMD Tutorial](<http://www.ks.uiuc.edu/Training/Tutorials/namd/namd-tutorial-unix-html/index.html>)
  * [Scripting in VMD](<http://www.ks.uiuc.edu/Training/Tutorials/vmd/tutorial-html/node4.html>). Some basic scripting skills in VMD is essential
