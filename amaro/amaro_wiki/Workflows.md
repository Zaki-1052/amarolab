# Workflows

Scientific workflow environments can be used to automate and repeat scientific protocols. They improve the efficiency of running certain procedures and, through their ability to replicate those procedures, improve reproducibility. As of June 2017, two workflows have been developed. 

## CADD (Computer-Aided Drug Design) Pipeline[edit](</mediawiki/index.php?title=Workflows&action=edit&section=1> "Edit section: CADD \(Computer-Aided Drug Design\) Pipeline")]

The CADD Pipeline workflow can be found on the NBCR website [[1]](<http://nbcr.ucsd.edu/cadd>). 

## AMBER GPU MD Workflow[edit](</mediawiki/index.php?title=Workflows&action=edit&section=2> "Edit section: AMBER GPU MD Workflow")]

This workflow is for running MD simulations with AMBER [[2]](<http://nbcr.ucsd.edu/products/workflow-distribution>). Downloads, tutorials, and instructions can be found on the NBCR website.
