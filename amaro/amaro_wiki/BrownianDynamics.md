# Brownian Dynamics[edit](</mediawiki/index.php?title=BrownianDynamics&action=edit&section=1> "Edit section: Brownian Dynamics")]

This page contains content involving Brownian Dynamics (BD) Simulations 

## Simulation Software[edit](</mediawiki/index.php?title=BrownianDynamics&action=edit&section=2> "Edit section: Simulation Software")]

### BrownDye[edit](</mediawiki/index.php?title=BrownianDynamics&action=edit&section=3> "Edit section: BrownDye")]

The main simulation software that we use for BD is BrownDye, which is developed in the McCammon Lab. [BrownDye main page](<http://browndye.ucsd.edu/>)

### APBS[edit](</mediawiki/index.php?title=BrownianDynamics&action=edit&section=4> "Edit section: APBS")]

BD simulations often require electrostatic fields. The Poisson Boltzmann equation solver [APBS](<http://www.poissonboltzmann.org/>) can be used to obtain these fields in .dx file format.
