# Multiscale Calculations using SEEKR[edit](</mediawiki/index.php?title=Multiscale&action=edit&section=1> "Edit section: Multiscale Calculations using SEEKR")]

## What is SEEKR?[edit](</mediawiki/index.php?title=Multiscale&action=edit&section=2> "Edit section: What is SEEKR?")]

SEEKR is a collection of scripts and templates designed to automate the preparation and analysis of ligand-receptor kinetic calculations that use both MD and BD, and combine the results using the theory of milestoning. 

## Download SEEKR[edit](</mediawiki/index.php?title=Multiscale&action=edit&section=3> "Edit section: Download SEEKR")]

The most up to date information regarding the SEEKR software can be found at [SEEKR main page](<https://amarolab.ucsd.edu/seekr/>)

For the latest version of SEEKR check out our [GitHub](<https://github.com/nbcrrolls/SEEKR/>)
