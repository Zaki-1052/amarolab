# LocalIT

Access to workstations and local software, printer, storagee 

## Software[edit](</mediawiki/index.php?title=LocalIT&action=edit&section=1> "Edit section: Software")]

All linux software is installed on shared file servers in /soft (for machines on CentOS or Scientific Linux) Ubuntu machines use the module command to load software. 

## Laptop software and licenses[edit](</mediawiki/index.php?title=LocalIT&action=edit&section=2> "Edit section: Laptop software and licenses")]

[OpenEye](</mediawiki/index.php/Openeye> "Openeye") may be downloaded from [[1]](<http://www.eyesopen.com>), and you can obtain the oe_license.txt from /soft/openeye/latest 

[Schrodinger](</mediawiki/index.php/Schrodinger> "Schrodinger") may be downloaded and installed on your laptop. Follow the instructions on connecting to the license server for different platforms. Or the settings may be entered in a license file. See /soft/schrodinger/latest to schrodinger for the example. Pymol is included with Schrodinger. 

[MOE](</mediawiki/index.php/Moe> "Moe") has a DVD somewhere on the desk by the kitchen area. There is one license file, and it's also in /soft/moe/latest. There is only one seat, so log off if you are not using it. 

MOE and Schrodinger need to connect to license servers, and these are regulated by firewall rules on the license servers. If you are off-campus please use VPN and that should give you access.
