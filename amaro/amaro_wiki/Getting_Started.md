## Tutorials[edit](</mediawiki/index.php?title=Getting_Started&action=edit&section=2> "Edit section: Tutorials")]

A useful tutorial for people new to Unix/Linux is the Unix Primer: <http://www.ks.uiuc.edu/Training/Tutorials/Reference/unixprimer.html>

Before starting each tutorial, remember to download all of the required tutorial input files (and, in some cases, the output files/results)! 

  1. VMD: <http://www.ks.uiuc.edu/Training/Tutorials/>


  *     * Please complete the first two tutorials: Using VMD and VMD Images and Movies Tutorial


  1. AutoDock: <http://autodock.scripps.edu/faqs-help/tutorial/using-autodock-4-with-autodocktools>
  2. Amber: <http://ambermd.org/tutorials/>


  *     * Please complete the B1-B5 tutorials


  1. NAMD: <http://www.ks.uiuc.edu/Training/Tutorials/>


  *     * Scroll down past VMD Tutorials. Please only complete the first one, NAMD Tutorial


  * If you're using Linux boxes, use command "ls /soft/linux/" to see a list of softwares. 



For example, opening vmd, you should do "/soft/linux/vmd-x86_64/1.9.0/vmd".
