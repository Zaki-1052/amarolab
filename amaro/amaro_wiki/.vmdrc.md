# .vmdrc

# VMDRC File[edit](</mediawiki/index.php?title=.vmdrc&action=edit&section=1> "Edit section: VMDRC File")]

You should create a .vmdrc file to optimize your usage of VMD. 

Navigate to your home directory by typing the following command: 
    
    
    cd
    

Then type the following command to open a new file: 
    
    
    gedit .vmdrc &
    

The ampersand (&) is designed to keep the terminal from hanging while gedit it open. Now copy and paste the following code into your open .vmdrc file: 
    
    
    
    # turn on lights 0 and 1
    light 0 on
    light 1 on
    light 2 off
    light 3 off
    
    # position the stage and axes
    axes location off
    stage location off
    
    # position and turn on menus
    menu main on
    menu graphics on
    menu files on
    after idle {menu tkcon on;
    	    menu tkcon move 1173 84}
    
    menu main move 5 196
    menu graphics move 5 455
    menu files move 1300 750
    display reposition 490 1025
    
    display projection orthographic
    display depthcue off
    
    color Type C orange
    color Element C purple
    
    
    

Then save and close.
